Updating a Mapping
==================

Function
--------

This API is used to update the information about a mapping.

URI
---

-  URI format

   PATCH /v3/OS-FEDERATION/mappings/{id}

-  URI parameters

   ========= ========= ====== ===========
   Parameter Mandatory Type   Description
   ========= ========= ====== ===========
   id        Yes       String Mapping ID.
   ========= ========= ====== ===========
